| 04—05 上海与重点省市比较 | 专利、主体、占比、排名、年度变化 | 第二节 |
| 06—08 技术方向比较 | 上海和重点省市五类技术结构 | 第二节 |
| 09—11 企业比较 | 企业规模、持续性、新进入 | 第三节 |

## 重要概念

- 专利数：识别为量子相关的发明授权专利数量。
- 创新主体数：第一申请人去重数量。
- 活跃年份数：主体在样本期内至少有一件相关专利的年份数。
- 新进入企业：样本期内首次出现的年份，不是企业真实成立或首次进入行业年份。
- CR5：前五家企业专利数占该地区企业专利总数的比例。
- 主要技术方向：该主体专利数量最多的main_category。
