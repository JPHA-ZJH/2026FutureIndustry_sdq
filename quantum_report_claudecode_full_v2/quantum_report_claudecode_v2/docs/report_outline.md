1. 量子计算
2. 量子通信与安全
3. 量子传感
4. 量子计算硬件
5. 量子基础概念

每个方向控制在200—500字。应说明基本原理、核心解决的问题、主要技术内容和现阶段应用或发展重点，尽量深入浅出，不堆砌名词，不写空泛意义。必须联网检索权威来源，并在来源表中记录。

“量子计算”和“量子计算硬件”需要区分：前者侧重计算系统、算法、软件和应用，后者侧重物理量子比特、芯片、操控和读出等底层实现。
