# 数据分析范围与边界

## 数据用途

- 专利明细表：技术方向、地区、主体类型和具体专利分析的主数据。
- 重建企业—年份表：企业年度活跃、进入和年度专利分析。
- 重建主体汇总表：企业和科研机构排名、活跃年份和主要技术方向分析。
- 原始两张主体汇总表：仅用于总量交叉核验，因为原始类型字段包含合作申请人的混合类型。

## 全国口径

“全国技术方向专利”指中国境内第一申请人的发明授权专利。外国第一申请人在中国获得的授权专利不计入全国地区和全国技术趋势，但保留在清洗数据中。

## 上海及省际口径

省级地区由第一申请人城市映射得到；城市名称先统一。无法映射的中国城市单独列入审计，不擅自归属。

## 企业口径

企业必须满足清洗后的`主体类型=企业`。原始字段名为firm的数据并不自动等于企业数据。

## 不能由当前数据回答的问题

- 产值、收入、就业、融资和估值；
- 市场份额、订单和产品销量；
- 合作网络和成果转化率；
- 技术先进性和商业化成熟度；
- 上海区级空间分布；
- 真实企业成立或进入量子产业的时间。
