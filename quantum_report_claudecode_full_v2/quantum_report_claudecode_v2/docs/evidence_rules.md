# 证据与表述规则

## A级：可直接写入正文

- `outputs/tables`中能够直接核对的数量、比例、排名和技术结构；
- 企业官网、政府网站、上市公司公告、正式年报；
- 国家部委、中国信通院、中国科学院等权威机构发布的报告。

## B级：需要谨慎表述

- 专利数量用于反映创新产出和技术布局；
- 活跃年份用于反映样本期内专利活动持续性；
- 企业集中度用于反映专利分布集中程度；
- 企业和科研主体技术结构差异用于识别可能的布局差异。

使用“数据显示”“从专利布局看”“在本研究口径下”等限定语。

## C级：不得仅凭当前数据写出

- “产业规模领先”“市场占有率高”“商业化成熟”；
- “技术达到国际领先水平”；
- “成果转化效率高/低”；
- “企业经营状况良好”；
- “某项政策导致专利增长”。

## 外部资料要求

- 关键事实至少有一个A级来源；争议性或重要判断尽量交叉核验。
- 不引用搜索结果摘要作为最终证据，应打开原网页核对。
- 所有外部资料记录到`report/source_log.csv`。
