# Claude Code 推荐完整对话

## 第一轮：运行、清洗和审计

> 请读取CLAUDE.md和prompts/01_run_clean_audit.md，运行python run_all.py。先检查城市标准化、第一申请人类型、三表一致性、输出表格和图形。发现代码问题时做最小修改并重新运行。暂时不要写报告。

## 第二轮：技术路线资料

> 请执行prompts/02_research_technical_routes.md。只使用权威来源，完成五类技术路线的资料卡和来源记录。暂时不要写完整报告。

## 第三轮：严格按提纲写初稿

> 请执行prompts/03_write_strict_draft.md。必须严格按照docs/report_outline.md的七节结构撰写，不得增加产业链和相关性分析。

## 第四轮：事实核验

> 请执行prompts/04_fact_check.md。逐项核验报告数字、企业类型、图表引用和外部事实，形成核验稿及问题清单。

## 第五轮：润色

> 请执行prompts/05_polish.md。在不改变章节结构和事实结论的前提下，形成政府咨询报告风格的终稿。
