# 版本说明：完整工作流 V2

本版本依据最新三份2021—2025年识别结果重构，重点完成以下调整：

1. **城市全国统一**：将“上海市/上海”“北京市/北京”等统一为标准城市名；使用全国城市—省份参考表进行映射，后续出现的新城市如未识别，将写入审计清单，不会被主观分配。
2. **第一申请人类型重判**：不再直接使用合作专利中混合的申请人类型；重新提取第一申请人，并依据第一申请人名称生成唯一的企业、高校、科研院所、机关团体、个人等类型。
3. **主体表重建**：以清洗后的专利明细重新生成企业—年份表和主体汇总表，避免原混合类型继续影响企业分析。
4. **技术分析口径收敛**：只分析量子计算、量子通信与安全、量子传感、量子计算硬件、量子基础概念五个技术方向；不分析相关性等级和产业链环节。
5. **第一节重写**：分为五类技术路线介绍和全国技术方向专利演进两部分；技术介绍须使用权威网络来源，每类200—500字。
6. **省际比较固定**：第二至第五节均以北京、安徽、江苏、浙江、广东作为上海的重点对比地区。
7. **企业案例扩充**：第六节包含上海3家代表性企业及外省市3—5家代表性企业，并区分专利数据事实与外部公开事实。
8. **图表更新**：生成22张分析表、13张高分辨率图，并修正年度横轴显示。
9. **严格章节约束**：Claude Code必须依照七节提纲写作，先清洗审计、再检索技术资料、再写初稿、再核验、最后润色。
