## 第七节 主要问题与政策建议

### （一）主要问题

所有问题必须能够回溯到前文表格、图形或权威外部材料。重点围绕总体位势、企业梯队、技术结构、科研主体与企业结构差异等方面归纳。

### （二）政策建议

政策建议应与问题一一对应，明确对象、环节和政策工具，避免“加强支持、加大投入”等泛化表述。不得把专利数据不能支持的产值、市场和转化结论作为政策依据。

## 数据口径与局限

报告末尾固定说明：

1. 仅统计识别为生物制造相关的发明授权专利；
2. 以第一申请人作为主要创新主体；
3. 城市经过标准化并映射到省份，未映射城市进入审计清单；
4. 合作专利的混合类型经过第一申请人名称规则重新识别；
5. 医疗机构单独分类；
6. 原始13类技术方向在正文归并为7条报告技术路线；
7. 专利数据不能替代经营、产值、融资和市场数据；
8. 最新年份可能受到授权滞后和数据库更新进度影响。
