# 分析范围与禁止越界事项

## 可以由数据直接分析

- 全国和各省市生物制造相关发明授权专利数量、创新主体数量及年度变化；
- 上海与北京、安徽、江苏、浙江、广东的总体比较；
- 企业数量、企业专利数量、活跃年份、新进入企业和专利集中度；
- 七条报告技术路线及原始13类技术方向的专利结构；
- 上海企业、高校、科研院所的专利数量和技术结构；
- 不同省市企业、高校、科研院所三类主体的专利占比差异；
- 代表性企业候选和代表性专利候选。

## 需要谨慎表述

- 2025年可能存在授权滞后和数据更新不完整；
- 样本期内首次出现不等于企业成立或实际进入产业的时间；
- 专利数量反映技术布局，不等于技术先进性；
- 企业排名是专利维度排名，不等于市场排名；
- 主体类型由名称规则判断，复核清单中的主体需要人工核验；
- 七条报告技术路线是对原始13类分类的咨询报告式归并。

## 现有数据不能直接分析

- 营业收入、产值、就业、融资、估值和市场份额；
- 产品量产、商业订单和实际应用规模；
- 产学研合作、专利共同申请网络和技术转移；
- 上海区级空间分布；
- 完整产业链交易关系；
- 成果转化率；
- 技术先进性或商业价值；
- 专利质量。被引证次数覆盖不足，识别得分也不能替代质量指标。
