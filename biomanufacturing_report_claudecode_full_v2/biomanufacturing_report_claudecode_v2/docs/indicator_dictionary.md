# 指标字典

| 指标 | 计算口径 | 注意事项 |
|---|---|---|
| 专利数 | 生物制造相关发明授权专利行数 | 不等于产业规模 |
| 创新主体数 | 第一申请人去重数 | 包含企业、高校、科研院所等 |
| 企业数 | 清洗后主体类型为企业的第一申请人数 | 不含待复核/其他机构 |
| 企业专利数 | 第一申请人为企业的专利数 | 合作专利仍按第一申请人口径 |
| 活跃年份数 | 主体在样本期内出现专利的年份数 | 不等于连续经营年限 |
| 新进入企业 | 样本期内第一次出现专利的企业 | 2021年是左截断起点，不能解释为真实首次进入 |
| CR5 | 前五家企业专利数占地区企业专利总数的比例 | 反映专利集中度 |
| HHI | 企业专利份额平方和 | 只反映专利维度集中度 |
| 技术路线占比 | 某技术路线专利数/相应范围全部专利数 | 正文使用七条报告技术路线 |
| 三类主体内部占比 | 企业/高校/科研院所某类专利数 ÷ 三类专利合计 | 图14专用，不包含医疗机构等 |
| 全国专利占比 | 地区专利数/国内已映射专利总数 | 境外申请人不进入省际比较 |
