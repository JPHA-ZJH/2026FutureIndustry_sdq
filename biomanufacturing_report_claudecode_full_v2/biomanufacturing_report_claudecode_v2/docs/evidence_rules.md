# 证据与写作规则

## 证据优先级

1. 本项目 `outputs/tables` 中的统计表；
2. 本项目 `outputs/figures` 中的图形；
3. 国家和地方政府部门正式文件；
4. 中国科学院、中国工程院、高校和国家级科研平台的正式资料；
5. 企业官网、上市公司公告和招股书；
6. 权威行业研究报告和同行评议论文；
7. 正式媒体仅用于补充，不作为唯一依据证明关键经营事实。

## 数字规则

- 每个数字必须能在表格中定位；
- 比例必须说明分母；
- 图14的占比仅以企业、高校和科研院所三类专利合计为分母；
- 不以 `biomanufacturing_score`、`core_score` 等识别得分表述专利质量；
- 不以缺失率较高的被引次数形成主要结论；
- 2025年趋势必须附数据完整性提示。

## 外部事实规则

- 技术路线介绍必须记录来源链接；
- 企业成立时间、产品、产能、融资、上市地位等必须有官网、公告或政府来源；
- 查不到可靠来源就删去，不根据模型记忆补写；
- 外部材料与专利结果不一致时，分别陈述，不能强行解释。

## 语言规则

使用“数据显示”“从专利维度看”“在当前样本中”等限定语。避免“全国领先”“形成完整产业链”“产业化成熟”等超出证据的结论，除非有明确权威材料支持。
