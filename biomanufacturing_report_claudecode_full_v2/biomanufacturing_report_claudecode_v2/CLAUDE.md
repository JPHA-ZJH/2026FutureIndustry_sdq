# Claude Code项目规则

本项目用于生成上海生物制造产业政府咨询报告。

## 工作顺序

必须按以下顺序执行：

1. 运行和审计数据；
2. 核验技术路线和政策资料；
3. 严格按固定章节写初稿；
4. 逐项事实核验；
5. 最后润色。

不得跳过分析代码，直接凭印象写报告。

## 不得擅自改变的口径

- 比较地区：上海、北京、安徽、江苏、浙江、广东；
- 企业依据清洗后的唯一主体类型；
- 医疗机构单独分类；
- 正文技术分析使用七条报告技术路线；
- 原始13类技术方向保留用于解释和附表；
- 不分析相关性等级和产业链环节；
- 图14的分母仅为企业、高校、科研院所三类专利合计；
- 被引证次数和识别得分不作为质量指标。

## 数据更新

替换 `data/raw` 中四个CSV后运行 `python run_all.py`。旧报告中的数字和企业名单不得直接沿用，必须基于新结果重写。
