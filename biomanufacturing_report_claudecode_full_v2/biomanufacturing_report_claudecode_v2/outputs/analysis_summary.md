# 生物制造分析结果摘要

- 当前数据覆盖至2025年，上海共有1,576件生物制造相关发明授权专利，涉及487个创新主体。
- 上海在国内省级地区中的专利数量排名为第6位，全国占比为5.67%。
- 上海企业主体398家，企业专利709件；高校和科研院所合计专利776件。
- 上海专利数量较多的技术路线为：生物过程与规模化、生物制造产品与应用、生物设计、读写与自动化工具。
