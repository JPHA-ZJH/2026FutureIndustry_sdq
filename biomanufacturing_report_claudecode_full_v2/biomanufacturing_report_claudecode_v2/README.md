# 上海生物制造产业咨询报告 Claude Code 工作包 V2

本项目用于基于生物制造相关发明授权专利数据，完成数据清洗、统计分析、图表生成、权威资料检索和咨询报告初稿撰写。

## 一、项目特点

1. 以专利明细为核心数据，重新生成企业—年份表和主体汇总表，避免原始合作申请记录中的混合申请人类型影响分析。
2. 自动统一“上海/上海市”“北京/北京市”等城市写法，并依据全国城市参考表映射省份；无法识别的中国城市进入审计清单，不进行主观猜测。
3. 根据第一申请人名称重新判断唯一主体类型，区分企业、高校、科研院所、医疗机构、机关团体、个人和其他机构。
4. 原始13类技术方向在分析底表中保留，咨询报告正文归并为7条技术路线。
5. 不分析相关性等级，不分析产业链环节，不把识别得分解释为专利质量。
6. 报告以上海企业为主，高校和科研院所作为科研支撑补充，并与北京、安徽、江苏、浙江和广东比较。
7. 第五节专门配置“不同地区企业—高校—科研院所专利占比差异图”和绝对数量图。

## 二、数据文件

原始数据位于 `data/raw/`：

- `biomanufacturing_patents_2021_2025_newsource.csv`：专利明细主数据；
- `firm_year_biomanufacturing_2021_2025_newsource.csv`：原始主体—年份汇总，用于一致性核验；
- `firm_biomanufacturing_2021_2025_newsource.csv`：原始主体汇总，用于一致性核验；
- `biomanufacturing_keyword_dictionary.csv`：生物制造关键词词典及原始技术方向。

更新正式数据时，保持文件名和字段名称不变，直接替换上述文件即可。

## 三、运行方法

在 VS Code 中打开项目根目录，在终端运行：

```bash
python -m pip install -r requirements.txt
python run_all.py
```

程序依次执行：

```text
原始数据读取
→ 城市标准化和省份映射
→ 第一申请人及唯一主体类型重建
→ 技术路线归并
→ 企业—年份表和主体汇总表重建
→ 数据审计
→ 统计表生成
→ 图表生成
```

运行完成后，优先查看：

1. `outputs/audit/data_cleaning_audit.md`
2. `outputs/analysis_summary.md`
3. `outputs/tables/`
4. `outputs/figures/`

## 四、Claude Code 使用顺序

依次向 Claude Code 提交：

1. `prompts/01_run_clean_audit.md`
2. `prompts/02_research_technical_routes.md`
3. `prompts/03_write_strict_draft.md`
4. `prompts/04_fact_check.md`
5. `prompts/05_polish.md`

完整对话说明见 `docs/claude_code_conversation.md`。

Claude Code 必须先运行代码和核验数据，再检索权威资料，最后严格按照 `docs/report_outline.md` 撰写报告。不得跳过数据审计直接写作。

## 五、报告固定结构

1. 全国生物制造技术路线及专利演进；
2. 上海生物制造创新发展的总体特征；
3. 上海生物制造企业主体与发展梯队；
4. 上海生物制造企业技术布局；
5. 上海高校和科研院所的技术支撑；
6. 上海及其他地区代表性生物制造企业；
7. 主要问题与政策建议。

第一节介绍7条技术路线及全国专利变化。第二至第五节末均设置重点省市比较。第六节选择上海3家和外省市3—5家代表性企业候选，并通过官网、政府文件或公告补充核验。

## 六、主要输出

- `data/clean/`：清洗后的专利、主体—年份和主体汇总数据；
- `outputs/audit/`：城市、主体类型、技术路线和一致性审计；
- `outputs/tables/`：25张分析底表；
- `outputs/figures/`：15张咨询报告图表；
- `outputs/draft/`：可直接查看的当前数据初稿；
- `report/`：初稿、技术路线说明和外部来源记录；
- `outputs/result_manifest.csv`：全部结果文件清单。

## 七、数据解释边界

- 专利数量反映技术创新和知识产权布局，不等同于产业产值、市场份额或商业化水平。
- 样本期内首次出现不等同于企业成立或首次进入生物制造。
- 主体结构占比不能直接解释为成果转化效率或合作强度。
- 最新年份可能受到授权滞后和数据库更新进度影响。
- 被引证次数当前覆盖率较低，不作为主要质量指标。
- 代表性企业名单是数据筛选候选，市场地位和实际业务必须通过权威来源核验。
