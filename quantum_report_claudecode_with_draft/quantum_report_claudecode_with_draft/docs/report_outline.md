# 量子信息章节建议提纲（经三份CSV可行性审核）

## 第一节 全国量子科技技术发展现状

1. 量子计算、量子通信、量子测量/传感三大方向的基本内涵。
2. 各方向当前技术成熟度与工程化进展。
3. 我国量子科技发展的总体特征。

**数据口径：** 本节主要使用权威外部资料，不使用三份CSV推断全国技术成熟度。篇幅控制在本章的10%—15%。

## 第二节 上海量子信息创新发展的总体特征

1. 上海量子相关发明授权专利和创新主体规模。
2. 多年正式数据下的年度变化、全国占比和排名变化。
3. 上海技术相关性结构：严格核心、核心口径、PNT候选。
4. 上海与北京、安徽、江苏、浙江、广东的规模和结构比较。

**当前2021年测试版：** 只能写2021年截面特征，不能写发展趋势。

## 第三节 上海量子信息企业主体与发展梯队

1. 明确识别为企业的主体数量和专利数量。
2. 多年正式数据下的活跃企业、新进入企业和持续创新企业。
3. 企业专利分布、CR3/CR5/CR10和HHI。
4. 核心、成长、专业和储备企业候选划分。
5. 与五个重点省市的企业数量、企业专利和集中度比较。

**限制：** 梯队仅代表专利创新表现，不代表营收、市场份额或综合企业实力。

## 第四节 上海量子信息企业技术与产业链布局

1. 量子计算、量子计算硬件、量子通信与安全、量子传感等技术方向。
2. 系统、硬件/芯片、软件算法、通信安全、精密测量、PNT和上游部件等环节。
3. 各方向的企业数量、专利数量和代表企业。
4. 上海优势方向与相对薄弱方向。
5. 与五个重点省市的技术结构和链条位置比较。

**限制：** `chain_position`是专利技术位置分类，不宜直接称为完整产业链规模。

## 第五节 上海高校和科研院所的技术支撑

1. 上海高校院所的专利规模和主要主体。
2. 高校院所的主要技术方向。
3. 企业与高校院所在技术结构上的对应关系。
4. 与五个重点省市的高校院所专利结构比较。

**限制：** 只写科研专利供给，不写合作效率、技术转移效率或成果转化率。

## 第六节 上海及其他地区代表性量子信息企业

1. 依据专利数据选择3家上海企业候选。
2. 逐家介绍专利规模、核心方向、活跃年份和代表性专利。
3. 从全国其他地区选择3—5家企业候选。
4. 比较技术方向、专利积累、硬件/系统/应用布局差异。
5. 对企业产品、融资和市场信息，仅使用企业官网、政府资料或权威媒体并注明来源。

## 第七节 主要发现、问题与政策建议

1. 先列数据事实，再给有限解释。
2. 问题与建议必须能回溯到表格、图形或权威来源。
3. 避免把单年授权专利、量子识别得分或专利数量等同于产业综合实力。
