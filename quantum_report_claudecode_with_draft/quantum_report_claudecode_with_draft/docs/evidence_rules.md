# 咨询报告证据与写作规则

## 一、证据分级

### A级：可直接作为结论依据

- 三份CSV经代码计算得到的表格和图形；
- 国家部委、地方政府、国家自然科学基金委、中科院、中国信通院等官方资料；
- 企业官网、正式年报、招股书和公告中关于企业自身的事实。

### B级：可作为辅助说明

- 权威行业协会、知名研究机构报告；
- 主流财经或科技媒体对公开事实的报道。

### C级：原则上不作为核心证据

- 自媒体、营销稿、无出处转载；
- 搜索摘要中的数字但未打开原文核验；
- 对专利数据的主观延伸推断。

## 二、每个判断的表达方式

优先使用：

- “数据表明……”
- “从专利识别结果看……”
- “在本报告口径下……”
- “与五个重点省市相比……”
- “这一结果可能反映……，但仍需结合经营和产业数据进一步验证。”

避免使用：

- “必然”“充分证明”“已经形成完整产业链”；
- “市场领先”“产业规模最大”，除非有相应经营或市场数据；
- 将`quantum_score`称为专利质量得分；
- 将单一年份下降解释为产业衰退。

## 三、外部资料来源顺序

1. 官方政策与科研机构；
2. 中国信通院等权威研究机构；
3. 企业官方信息；
4. 主流媒体交叉核验。

全国技术现状可优先检索中国信通院《量子信息技术发展与应用研究报告（2025年）》和《量子计算发展态势研究报告（2025年）》的官方发布页，并结合国家自然科学基金委量子研究计划及政府部门资料。每一项动态事实都需在`templates/source_log.csv`登记。
