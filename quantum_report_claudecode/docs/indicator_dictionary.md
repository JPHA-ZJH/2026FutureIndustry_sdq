# 指标说明

| 指标 | 计算方式 | 可解释内容 | 不可解释内容 |
|---|---|---|---|
| 量子相关专利数 | 识别为量子相关的发明授权专利记录数 | 技术创新产出和布局规模 | 产值、市场规模 |
| 创新主体数 | 第一申请人去重数 | 参与量子创新的主体广度 | 实际从业企业总量 |
| 明确企业数 | 经申请人类型和名称规则明确识别为企业的第一申请人数 | 保守口径下企业创新主体 | 全部量子企业 |
| 严格核心专利 | 高相关+中相关 | 与核心量子词匹配较强的专利 | 专利法律质量、商业价值 |
| 核心口径专利 | 高相关+中相关+低相关/待复核 | 较宽核心量子口径 | 必然属于产业化量子技术 |
| PNT候选 | 量子计量/时间频率/PNT候选 | 原子钟和时间频率相关技术储备 | 可与量子计算、通信直接等同 |
| quantum_score | 关键词规则得分 | 量子相关性强弱 | 专利质量、价值或先进性 |
| CR5 | 前五家明确企业专利占企业专利总量比例 | 企业专利集中程度 | 市场集中度 |
| HHI | 企业专利份额平方和 | 企业专利分布集中程度 | 销售或市场集中度 |
| 新进入企业 | 该企业首次出现量子专利的年份 | 专利口径下新进入 | 企业成立或正式进入市场时间 |
| 持续创新 | 多个年份出现量子专利 | 专利活动持续性 | 连续研发投入或经营稳定性 |
