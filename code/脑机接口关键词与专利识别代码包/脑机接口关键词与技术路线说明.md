# 脑机接口关键词、技术路线与专利识别说明

## 1. 文件用途与口径

本文件与配套关键词词典、匹配器、企业汇总代码共同用于识别中文专利文本中的脑机接口技术，并按照技术领域、细分方向、产业板块和核心程度进行结构化标注。

- 识别文本：专利名称、摘要文本、主权项内容。
- 识别对象：2021—2025 年中国专利数据库中的“发明授权”。
- 申请人统计：按照中文分号“；”取第一申请人，并同步提取第一申请人类型、地区和城市；兼容少量英文分号。
- 关键词语言：以中文为主体，保留中文科技文本中常见的 BCI、EEG、ECoG、SEEG、LFP、SSVEP、P300、DBS、fNIRS、fUSI 等缩写，不机械建立一套与中文完全重复的英文词典。
- 输出口径：不再划分“高相关、低相关、待复核”，仅输出连续的 1—5 分核心程度；原始证据累计分另行保留。
- 默认纳入门槛：最终核心程度得分不低于 1 分。外围应用得分较低，核心接口、核心器件和关键算法得分较高。

第二份上传附件的本地副本存在 PDF 结构截断，无法完整解析。本次梳理使用中国信息通信研究院官网发布的同名正式版进行交叉核对，不改变报告来源和分析口径。

## 2. 总体技术路线

两份报告共同支持以“脑感知—脑调控”为横向主线、以接口侵入程度和技术成熟阶段为纵向维度的技术体系。

### 2.1 脑感知路线

脑感知是将脑活动转换为可分析、可解码信号的技术路线，主要包括：

1. 电信号感知
   - 头皮脑电：EEG、高密度脑电、干电极、湿电极、入耳式脑电。
   - 皮层表面信号：ECoG、微型 ECoG、硬膜下/硬膜外电极。
   - 深部信号：SEEG、LFP、Spike、单神经元放电。
   - 关键环节：低噪声放大、模数转换、同步采集、伪迹抑制、无线传输。

2. 磁信号感知
   - 脑磁图 MEG。
   - SQUID 超导量子干涉器件。
   - OPM 光泵原子磁力计、可穿戴脑磁图。

3. 光学和血流动力学感知
   - fNIRS 功能近红外光谱。
   - fMRI/rt-fMRI 实时功能磁共振反馈。
   - 血氧和血红蛋白浓度变化检测。

4. 超声感知
   - fUSI 功能超声成像。
   - 超快超声、脑血流动力学成像、微型超声脑接口。

5. 多模态感知
   - EEG-fNIRS、EEG-fMRI、脑电—眼电、脑电—肌电等融合。
   - 电、磁、光、超声多源同步采集与信息融合。

### 2.2 脑调控路线

脑调控是将外部信息或控制策略编码为刺激并作用于神经系统的技术路线，主要包括：

1. 电刺激
   - 植入式：DBS/aDBS、皮层微刺激、脊髓电刺激、外周神经刺激。
   - 无创式：tDCS、tACS、tRNS、高精度经颅电刺激。
   - 发展重点：采集—刺激一体化、刺激伪迹消除、参数自适应和闭环控制。

2. 磁刺激
   - TMS、rTMS、TBS、模式化磁刺激 pTMS。
   - 发展重点：脑电引导、个体化靶点和闭环磁刺激。

3. 超声调控
   - TUS、LIFU、经颅聚焦超声、影像引导超声调控。

4. 光调控
   - 光遗传学、tPBM 经颅光生物调节、近红外调控、光电一体神经接口。

5. 神经反馈与组合调控
   - EEG/fNIRS/fMRI 神经反馈。
   - 电磁、磁声、电光等多模态组合调控。
   - 依据实时脑状态自动调整刺激的感知式、闭环式调控。

### 2.3 按侵入程度划分的接口路线

1. 侵入式：皮层内电极、犹他阵列、Neuropixels、柔性微丝、全植入无线接口。
2. 半侵入/微创式：ECoG、硬膜外/硬膜下电极、皮层表面柔性电极。
3. 血管介入式：血管内支架电极、脑静脉窦电极、Stentrode。
4. 非侵入式：EEG、MEG、fNIRS、fMRI、fUSI 以及相应可穿戴系统。
5. 混合式：多模态 BCI、脑电—肌电、脑电—眼电等。

### 2.4 按融合阶段划分的发展方向

1. 1.0 阶段：脑信号独立采集或单向刺激，感知和调控相对分离。
2. 2.0 阶段：交互式脑感知、感知式脑调控，形成实时反馈或闭环调节。
3. 3.0 阶段：脑感知与脑调控深度融合，实现双向、高带宽、自适应的人机智能协同。

## 3. 技术领域与细分方向

### 3.1 上游核心器件与材料

- 神经电极与探针：犹他阵列、柔性微丝、Neuropixels、ECoG 电极、SEEG 电极、干湿脑电电极、血管内支架电极。
- 电极材料：铂铱、铱氧化物、石墨烯、碳纳米管、PEDOT:PSS、导电水凝胶、聚酰亚胺、Parylene。
- 界面与可靠性：低阻抗、高电荷注入容量、生物相容、抗炎/抗胶质瘢痕、长期稳定、气密封装。
- 芯片与电路：神经模拟前端、ADC、高通道采集、低功耗处理、无线通信、片上解码、采集刺激一体化、存算传一体。
- 微纳制造与封装：MEMS、光刻、薄膜沉积、晶圆级制造、柔性互连和植入体封装。
- 植入辅助：手术机器人、立体定向、神经导航、脑血管避让、微创递送。

### 3.2 中游采集、处理、编解码与调控

- 信号采集：多通道同步记录、低噪声放大、滤波、抗干扰、压缩和低时延传输。
- 预处理：工频/运动/眼电/肌电伪迹抑制、重参考、ICA、小波去噪。
- 特征提取：时频特征、功率谱、ERD/ERS、CSP/FBCSP、脑网络连接、源定位、Spike sorting/Kilosort。
- 实验范式：P300/ERP、SSVEP/VEP、运动想象/SMR、SCP、听觉诱发、言语、情感、被动式和异步 BCI。
- 神经解码：运动意图、轨迹、抓握、步态、语音、文字、手写、视觉语义、认知和情绪状态解码。
- 智能算法：SVM、CNN、RNN/LSTM、Transformer、图神经网络、迁移学习、域适应、自监督学习、在线自适应和边缘推理。
- 神经编码：触觉、视觉、听觉、体感和本体感觉编码，人工感觉反馈。
- 闭环调控：DBS、脊髓/外周电刺激、TMS、超声、光遗传和多模态调控。

### 3.3 下游应用

- 医疗康复：卒中、脊髓损伤、瘫痪和手功能康复，BCI-FES、康复机器人、外骨骼。
- 功能替代与辅助交流：ALS/闭锁综合征交流、脑控打字、语音合成、神经义肢、视觉/听觉/触觉假体。
- 神经疾病治疗：帕金森、癫痫、抑郁、强迫症、疼痛和意识障碍等场景中的闭环诊疗。
- 消费与教育：专注力训练、认知训练、睡眠改善、娱乐、游戏和虚拟现实。
- 工业交通：疲劳和认知负荷监测、智能座舱、高危作业、航空航天和人因工程。
- 脑控终端：光标、键盘、轮椅、机械臂、机器人、无人机、智能家居和物联网设备。
- 科研和临床平台：神经数据平台、算法平台、测试验证、临床转化和标准评价。

### 3.4 安全、伦理、标准与临床转化

- 植入安全：感染、免疫反应、胶质瘢痕、组织损伤、取出和长期可靠性。
- 电气与无线安全：电荷平衡、温升、电磁兼容、无线暴露和电池安全。
- 数据与网络安全：脑数据隐私、神经数据脱敏、接口认证、抗攻击和安全传输。
- 神经伦理：知情同意、认知自由、人格同一性、增强公平性和责任归属。
- 标准与评价：术语、数据格式、互操作、性能测试、临床试验、注册审评和上市后监测。

## 4. 关键词核心程度评分

| 得分 | 含义 | 典型内容 |
|---:|---|---|
| 5 | 构成完整脑机系统、核心范式或核心闭环 | 脑机接口系统、侵入式 BCI、SSVEP 脑机接口、双向接口、采集刺激一体化、闭环 DBS、神经语音解码 |
| 4 | 关键器件、关键感知/调控路线或核心算法 | 柔性神经电极、高通道采集芯片、ECoG/SEEG、运动意图解码、Transformer 脑电解码、TMS/超声闭环调控 |
| 3 | 重要支撑环节、医疗康复或研究平台 | 信号预处理、伪迹去除、植入导航、脑机康复、脑控机械臂、测试验证平台 |
| 2 | 外围应用或与脑机结合较弱的终端场景 | 疲劳监测、智能座舱、游戏、教育、智能家居、普通状态监测 |

- `brain_computer_interface_score_raw`：所有有效关键词得分的累计证据强度。
- `brain_computer_interface_score`：专利最终核心程度，取有效命中中的最高等级并考虑负向规则，范围为 0—5。
- `core_score`：为兼容后续统计而保留，与最终核心程度一致。
- 主技术领域、细分方向、产业板块、命中词、无上下文失活词和负向词。

## 5. 上下文识别原则

关键词分为两类：

1. 独立领域词或核心词：如“脑机接口”“侵入式 BCI”“P300 脑机接口”“闭环脑机接口”，单独出现即可作为有效证据；但仅有“脑机接口”这一领域锚点时只给 1 分，不会自动抬高外围专利。
2. 上下文词：如“柔性电极”“干电极”“卷积神经网络”“Transformer”“机械臂”“经颅磁刺激”。这些词单独出现不会判为脑机接口；只有在文本已存在脑机接口语境，或同时存在脑/神经信号和解码、交互、反馈、控制语境时才激活。

上下文成立后，通用词不因其字面通用而被一律降分，而是依据其在脑机系统中的真实技术地位评分。例如：

- “脑机接口采用高密度柔性电极阵列”中的“柔性电极”是上游核心器件，可得 4 分。
- “采集脑电并用 Transformer 解码运动意图控制机械臂”中的 Transformer 是中游核心算法，可得 4 分。
- “普通卷积神经网络接口”“计算机网络接口控制器”不满足脑机上下文，得 0 分。
- “普通脑电图用于癫痫诊断”“脑电用于睡眠监测”若没有人机交互、意图解码、反馈控制或脑机接口表述，不纳入。
- 单独的 TMS、DBS、tDCS 等常规神经调控治疗不自动纳入；只有脑机、脑信号引导、感知反馈或闭环调节明确成立时才纳入。
