# 6G关键词、技术路线与专利识别说明

## 1. 文件用途与识别口径

本说明与配套关键词词典、专利匹配器、企业汇总代码共同用于识别中文专利文本中的第六代移动通信（6G/IMT-2030）技术，并标注技术领域、细分方向、产业板块和核心程度。

- 数据范围：2021—2025年中国专利数据库。
- 专利类型：仅保留“发明授权”。
- 识别文本：`专利名称 + 摘要文本 + 主权项内容`。
- 申请人统计：按照中文分号“；”提取第一申请人、第一申请人类型、第一申请人地区和第一申请人城市，同时兼容少量英文分号。
- 词典语言：以中文为主，保留中文通信技术文本中常见的6G、IMT-2030、RIS、ISAC、NTN、OAM、OTFS、AI-RAN等缩写；不另建一套与中文完全重复的英文关键词典。
- 相关性表达：不划分“高相关、低相关、待复核”，统一使用1—5分核心程度评分，并保留累计证据分。
- 默认门槛：最终得分不低于1分即进入6G专利结果；可在企业脚本中调整 `MIN_SIXG_SCORE`。

## 2. 总体技术框架

两份报告共同表明，6G并非单纯提高无线速率，而是形成通信、感知、计算、智能、安全和空天地网络深度融合的新型数字基础设施。主要技术路线可以概括为以下十二类。

### 2.1 新频谱与新传播机制

1. 太赫兹和亚太赫兹通信
   - 0.1—10 THz频段、100 GHz以上频段、D波段和G波段。
   - 太赫兹信道测量、分子吸收、遮挡、雨衰、波束偏斜和近场传播。
   - 太赫兹收发架构、太赫兹天线阵列、光生太赫兹和光电融合太赫兹。

2. 可见光和光无线通信
   - VLC、LiFi、自由空间光通信FSO、红外/紫外无线通信。
   - 高速LED/Micro-LED、激光二极管、光电探测、光射频融合。

3. 低中高全频谱融合
   - Sub-6 GHz、厘米波、毫米波、太赫兹协同。
   - 多频段聚合、跨频段联合传输和动态频谱调度。

### 2.2 增强型无线空口

1. 调制编码
   - 统一编译码架构、Polar码、QC-LDPC、短码长编码、神经信道编码。

2. 新型波形
   - OTFS、AFDM、FTN、SEFDM/SEFFM、OVXDM、FBMC和通感一体波形。

3. 新型多址
   - NOMA、RSMA、SCMA、PDMA、MUSA、免授权接入和AI多址接入。

4. 超大规模天线
   - XL-MIMO、ELAA、连续孔径、超大规模口径阵列、分布式超大规模MIMO。
   - 近场MIMO、球面波信道、近场波束聚焦。

5. 无蜂窝网络
   - Cell-free Massive MIMO、用户中心虚拟小区、分布式接入点和协作多点。

6. 全双工
   - 带内全双工、同频同时收发、模拟/射频/数字域自干扰抵消。

### 2.3 新物理维度无线传输

1. 可重构智能表面
   - RIS/IRS、STAR-RIS、有源/无源RIS、编码超表面、全息超表面。
   - 相位控制、级联信道估计、反射系数优化、RIS辅助通信/感知/定位。

2. 轨道角动量
   - OAM波束、涡旋电磁波、模态复用、模态分选和太赫兹OAM。

3. 智能全息无线电
   - IHR、全息MIMO、射频空间谱全息、连续孔径全息天线和波场合成。

4. 微波光子融合
   - 微波光子阵列、相干光上变频、光域波束赋形和射频光子集成。

### 2.4 通信感知一体化

1. 通感系统：ISAC、JCAS、雷达通信一体化、通信定位感知一体化。
2. 通感波形：联合波形、预编码、波束赋形、频谱共享和功率分配。
3. 无线感知：目标检测、动作识别、呼吸心跳检测、射频成像和微多普勒感知。
4. 高精度定位：厘米/毫米级定位、三维定位、RIS/太赫兹辅助定位。
5. 感知辅助通信：无线环境地图、信道知识地图、感知辅助波束和资源调度。

### 2.5 AI原生空口与AI原生网络

1. AI for Network
   - AI-RAN、AI信道估计、AI波束管理、AI资源调度、神经接收机。
   - 端到端学习收发机、深度学习物理层、多智能体协同。

2. 内生智能
   - AI原生空口、内生智能网络、网络自运维、自检测、自修复和自演进。

3. 分布式学习
   - 无线联邦学习、分割学习、群智协同、隐私保护联邦学习。

4. Network for AI
   - 泛在AI服务、端边云AI协同、分布式推理、智能体通信和大模型传输。

### 2.6 语义通信和任务导向通信

1. 语义通信：语义编码/解码、语义信息论、多模态语义通信、生成式语义通信。
2. 任务导向通信：面向任务/目标/意图的通信、任务级传输、机器语义通信。
3. 空中计算：AirComp、无线聚合计算、联邦学习空中聚合和函数计算传输。

### 2.7 新型网络架构

1. 分布式自治：去中心化、用户中心、场景定制、需求驱动和自生长架构。
2. 云原生和服务化：微服务核心网、服务化RAN、容器化网元和按需编排。
3. 开放可编程：Open RAN、O-RAN、SDN、NFV和可编程数据面。
4. 网络切片：端到端切片、算网切片、通感切片和跨域切片。
5. 确定性网络：有界时延、抖动、丢包和跨域确定性服务。
6. 数据服务：数据驱动架构、数据随路计算、数据编排和网络数据治理。

### 2.8 数字孪生网络和算网融合

1. 数字孪生网络DTN
   - 物理网络—虚拟孪生体实时映射。
   - 网络建模、仿真、预测、闭环优化和控制。

2. 算力感知网络
   - 移动算力网络、算力路由、算力度量、算力服务调度。
   - 通信—计算—存储融合和端边云网协同。

### 2.9 空天地海一体化

1. 融合架构
   - SAGIN、NTN、星地融合、天地一体、空天地海一体。

2. 卫星网络
   - LEO/MEO/GEO多轨协同、星地融合空口、卫星蜂窝融合。
   - 星载核心网、星载基站、再生式/透明转发载荷、星上计算。
   - 卫星随机接入、波束跳变、多普勒补偿、移动性管理和多星切换。

3. 星间和星地链路
   - 星间激光、星地激光、太赫兹/毫米波星地链路和多星协同。

4. 终端和空基网络
   - 手机直连卫星、卫星物联网终端、HAPS、无人机基站和低空智联网。

### 2.10 芯片、器件、天线与材料

1. 太赫兹器件
   - 混频器、倍频器、功率放大器、低噪声放大器、振荡器、调制器、探测器。

2. 半导体和光子技术
   - InP、SiGe、GaN、CMOS、石墨烯、二维材料、硅光和光子集成。

3. 基带和AI芯片
   - 6G基带、神经网络基带、片上AI接收机、通感一体芯片、通信计算融合芯片。

4. 天线
   - 太赫兹阵列、相控阵、AiP、连续孔径、透镜天线和全息天线。

5. RIS器件
   - PIN/变容二极管、MEMS、液晶、石墨烯超表面、RIS控制芯片。

6. 高频材料与封装
   - 低损耗覆铜板、LCP、PTFE、低介电材料、高频陶瓷、异质集成和晶圆级封装。

### 2.11 绿色通信和内生安全

1. 绿色通信
   - 网络能效优化、碳感知网络、按需唤醒、零比特零瓦特。
   - 无源物联网、环境反向散射、能量采集和SWIPT。

2. 安全可信
   - 内生安全、多模信任、零信任、网络韧性和动态防御。
   - 隐私计算、物理层安全、后量子密码、量子密钥分发和区块链。

### 2.12 测试标准和应用

1. 测试验证
   - 6G原型、试验网、测试床、仿真平台、信道探测仪、太赫兹OTA测试。
   - ITU IMT-2030框架、3GPP Rel-20研究和Rel-21规范工作。

2. 应用
   - 全息通信、云XR、多感官交互、触觉互联网和智慧交互。
   - 工业数字孪生、无线工业总线、机器人协作和智能电网。
   - 车联网、自动驾驶、低空经济、无人机、远程医疗和应急通信。
   - 海量物联、智能体互联、具身智能和环境监测。

## 3. 产业链分类

### 3.1 上游

- 材料：高频覆铜板、LCP/PTFE、低介电材料、化合物半导体、超材料。
- 芯片器件：6G基带、射频前端、太赫兹器件、光子芯片、RIS控制器。
- 天线与模组：超大规模阵列、太赫兹天线、AiP、RIS阵列、卫星通信模组。
- 仪器仪表：信道探测、太赫兹信号源、频谱/网络分析、OTA和系统仿真设备。
- 通信设备：6G基站、核心网、星载载荷、卫星终端、光承载设备。

### 3.2 中游

- 无线空口：编码、波形、多址、MIMO、全双工、波束管理。
- 融合技术：通感一体、AI原生、语义通信、RIS/OAM/全息无线电。
- 网络架构：云原生、分布式自治、确定性网络、数字孪生网络、算力网络。
- 空天地网络：NTN、星地融合、卫星接入、移动性和路由。
- 运营与平台：网络运营商、云网服务、试验平台、标准和测试认证。

### 3.3 下游

- 终端：手机、XR、车载、工业、卫星、物联网和可穿戴终端。
- 行业应用：工业、交通、低空、医疗、教育、能源、城市治理和应急救援。
- 消费应用：全息、多感官、智慧交互、数字人和智能体互联。

## 4. 1—5分核心程度评分

| 得分 | 含义 | 示例 |
|---:|---|---|
| 5 | 明确的6G核心系统、代表性融合路线或核心范式 | 6G通信系统、6G太赫兹通信、6G通感一体、AI原生空口、语义通信、6G空天地一体、6G内生安全 |
| 4 | 6G关键器件、核心空口、网络架构和算法 | 太赫兹收发芯片、RIS、XL-MIMO、OAM、OTFS、数字孪生网络、算力感知网络、星载核心网 |
| 3 | 重要设备、测试支撑、终端或较强应用结合 | 6G终端、手机直连卫星、确定性网络、测试平台、高频封装、网络切片 |
| 2 | 明确属于6G但主要贡献位于应用端 | 6G远程医疗、6G车联网、6G智慧工厂、6G全息会议、6G低空应用 |
| 1 | 仅出现6G领域锚点或外围配套 | 单独的“6G”表述、设备外壳、保护套、培训、项目管理等 |

输出同时包含：

- `sixg_score_raw`：所有有效命中词的累计证据分。
- `sixg_score`：最终1—5分核心程度；0表示未通过识别。
- `matched_terms`：有效命中词。
- `inactive_terms_no_context`：因缺少6G上下文而未激活的通用技术词。
- 主技术领域、细分方向、产业板块和来源类型。

## 5. 上下文识别和消歧

### 5.1 领域锚点不自动给高分

“6G”“IMT-2030”“B5G”等可确认研究对象，但本身只代表领域归属。仅出现“6G”的专利得1分；只有出现核心系统、关键器件、空口算法或融合路线时才升至4—5分。

例如：

- “一种6G设备保护套”得1分。
- “一种6G太赫兹收发芯片及天线阵列”得4分。
- “一种6G通信系统及其AI原生空口方法”得5分。

### 5.2 通用技术词需要上下文

RIS、MIMO、波束赋形、联邦学习、卫星通信、数字孪生、太赫兹器件等技术并非只服务6G，因此单独出现不会自动判为6G。其激活条件为：

1. 文本存在6G、IMT-2030、B5G等明确上下文；或
2. 无明确代际词，但至少同时出现三个相互独立的6G候选技术簇，如“太赫兹通信 + RIS + 通感一体化”。

第二种方式用于识别少量未写出“6G”、但技术组合高度指向6G的专利，并采用保守门槛降低误判。

### 5.3 典型排除

- “5G-A网络采用RIS增强覆盖”：没有6G上下文，不纳入。
- “一种太赫兹探测器”：只有单一器件词，不纳入。
- “普通数字孪生生产线”：没有6G网络语境，不纳入。
- “6 GHz无线局域网”：6 GHz不是6G移动通信，不纳入。
- “称取6 g药物”：属于重量/剂量表达，不纳入。
