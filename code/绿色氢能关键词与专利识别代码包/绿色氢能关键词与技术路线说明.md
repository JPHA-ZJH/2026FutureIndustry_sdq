# 绿色氢能关键词、技术路线与评分说明

## 1. 交付内容与口径

本文件与 `green_hydrogen_keyword_dictionary.csv`、`green_hydrogen_matcher_v1_optimized_region_newsource.py`、`green_hydrogen_firm_2021_2025_newsource.py` 配套使用。

- 识别对象：与绿色氢能的生产、储存、输送、加注、转化利用、安全检测和系统集成直接相关的中国发明授权专利。
- 文本字段：专利名称、摘要文本、主权项内容。
- 企业口径：按中文分号“；”取得第一申请人，同时提取第一申请人类型、地区和城市。
- 评分口径：最终得分为命中技术中最高的核心程度（1—5分），原始证据累计分单列；不再划分高相关、低相关和待复核。
- 词典语言：中文为主体；ALK、PEM、AEM、SOEC、MEA、LOHC、PtX等在中文文本中常见的缩写直接保留，不机械复制整套英文词典。

## 2. 主要技术路线与细分方向

### 2.1 可再生能源制氢与系统集成

- 电氢耦合：风电制氢、光伏制氢、风光制氢、深远海制氢、离网/并网制氢、源网荷储氢、制储输用一体化。
- 电解路线：碱性电解（ALK/AWE）、质子交换膜电解（PEM/PEMWE）、阴离子交换膜电解（AEM/AEMWE）、固体氧化物电解（SOEC）、可逆固体氧化物电池（rSOC）。
- 复合与前沿路线：碱性—PEM混合电解、海水/废水电解、光电化学制氢、光催化制氢、热化学循环制氢、人工光合作用制氢。
- 动态运行：宽负荷、快速启停、波动电源适应、群控、功率分配、能量管理、寿命衰减和故障诊断。

### 2.2 制氢关键材料、部件与辅机

- 电解槽/电堆：电解小室、极板、极框、端板、拉杆和密封结构。
- 膜与电极：PPS隔膜、复合隔膜、质子交换膜、阴离子交换膜、膜电极（MEA）。
- 催化与传输层：HER/OER催化剂、铱/铂基催化剂、镍铁及非贵金属催化剂、多孔传输层（PTL）、气体扩散层（GDL）、钛毡、镍网、双极板。
- 系统辅机：整流电源、电解液循环、气液/氢氧分离、氢气纯化与干燥、去离子水、撬装BOP。

### 2.3 储氢

- 高压气态：35/70 MPa储氢、III/IV型瓶、碳纤维缠绕瓶、长管拖车和管束集装箱。
- 液氢：氢液化、正仲氢转化、低温储罐、车载液氢、蒸发气（BOG）和零蒸发控制。
- 地下储氢：盐穴、枯竭油气藏、含水层和岩穴储氢。
- 固态/吸附：金属氢化物、镁基/稀土储氢合金、硼氢化物、MOF和多孔碳材料。
- 化学载体：液态有机氢载体（LOHC）、氨、甲醇、甲酸和氨硼烷。

### 2.4 输氢与加氢

- 管道：纯氢长输管道、跨区域管网、海底管道、天然气掺氢及氢气分离。
- 装备：隔膜/液驱/离子液体压缩机、膨胀机、氢用阀门、流量计、调压器和密封。
- 加氢基础设施：加氢母/子站、液氢站、制加氢一体站、撬装站、加氢机/枪、预冷和顺序控制。

### 2.5 氢能利用与氢基衍生品

- 燃料电池：PEMFC、SOFC、AFC、可逆燃料电池及其电堆、膜电极、催化剂、双极板、扩散层和BOP。
- 氢动力与发电：氢/掺氢燃气轮机、氢内燃机、氢锅炉、氢能发电、热电联供和长时储能。
- 工业脱碳：氢基直接还原铁（DRI）、富氢高炉、氢基竖炉、炼化替代。
- 氢基衍生品：绿色合成氨、绿氨燃料、氨裂解；绿色甲醇、电制航煤、e-fuel/eSAF。
- 终端交通：重卡、公交、船舶、航空、无人机和轨道交通。

### 2.6 安全、材料、检测与认证

- 氢脆、氢致开裂、氢渗透、氢相容材料、输氢管线钢和氢用密封。
- 泄漏/火焰检测、通风防爆、安全联锁、放散和风险评估。
- 氢品质、在线分析、计量、绿氢认证、碳足迹和生命周期评价。

## 3. 评分与上下文逻辑

| 得分 | 含义 | 典型内容 |
|---|---|---|
| 5 | 核心范式或核心制氢路线 | 绿氢、可再生能源电解水制氢、ALK/PEM/AEM/SOEC、电氢耦合 |
| 4 | 关键装备、储运或工业脱碳路线 | 电解堆、膜电极、液氢、盐穴储氢、输氢管道、氢冶金、绿氨/绿醇 |
| 3 | 关键部件、辅机、安全或能源转换子系统 | PTL/GDL、整流器、压缩机、加氢站、氢安全、氢脆 |
| 2 | 终端应用、标准认证等外围环节 | 氢能交通、绿氢认证、生命周期评价 |
| 1 | 极外围但在明确绿色氢能上下文中成立的技术 | 词典保留扩展空间；当前版本尽量不靠模糊产品词单独触发 |

通用词不会单独触发。例如“催化剂、质子交换膜、压缩机、管道、双极板、燃料电池部件”需要氢能与绿色能源语境，或同一文本中已有强专用词。只要上下文确认其属于绿色氢能，仍按技术本身的重要性给3—4分，而不是因为词本身通用就机械降为低分。

程序对煤制氢、焦炉煤气制氢、天然气重整制氢、工业副产氢和灰氢设置排除/降分规则；如果同一专利明确涉及绿氢耦合、替代或比较，则保留但降低1分。过氧化氢、催化加氢脱硫、氢化油、氢键等非氢能含义直接排除。

## 5. 使用建议

正式统计前建议抽取至少300—500条候选专利分层复核，分别观察1—5分样本的准确率。程序默认 `MIN_GREEN_HYDROGEN_SCORE=1`，便于保留低相关但仍有效的专利；如研究需要更保守的核心样本，可改为2或3，但不应改变词典得分本身。
