# 生物制造关键词、技术路线与专利识别说明

## 一、报告范围与客观判断

可以依据两份报告形成较稳定的生物制造技术分类、细分方向和专利关键词词典。

- 《2024生物制造行业研究报告》明确给出生物制造的产业边界：以工业生物技术为核心，利用微生物、酶和生物过程，通过工业发酵、催化、合成、生物炼制和生物过程工程实现规模化生产；同时把合成生物学视为生物制造最具潜力的路径，并明确基因测序、基因合成、基因编辑、DBTL、代谢工程、元件/线路工程以及医药、材料、化学品、食品、能源等应用。
- SETR 2026 用户指定的标注页39—57中，标注页39—55为“Biotechnology and Synthetic Biology”章节；标注页56为章节过渡，标注页57已经进入密码学章节。因此本词典只吸收标注页39—55的实质相关内容，不把标注页57的密码学内容并入生物制造。
- SETR的主要增量是虚拟细胞、生成式生物学、基因组基础模型、分布式/按需生物制造、DNA分布式读写、活体和嵌入式生物技术、细胞尺度工程、人工合成细胞、3D生物打印、电生物合成、DNA数据存储以及生物安全。

词典以中文为主体，但不严格限定为汉字。中文专利和中文研报中经常使用的 DBTL、CRISPR、MAGE、TALEN、NGS、FACS、CFPS、eBio、PHA、PLA、SAF、CAR-T、HMO、iPSC 等缩写会直接保留；不再建立与中文词典一一重复的完整英文版本。

## 三、关键词核心程度和专利识别规则

### 1. 先确认“生物工程+生产制造”语境

- “生物制造、合成生物学、工业生物技术、细胞工厂、DBTL、分布式生物制造、电生物合成”等专用词可以独立触发。
- 基因编辑、CRISPR、发酵、酶、蛋白、微生物、细胞培养、纯化、具体化学品或食品等通用词，必须同时具有生物工程语境和制造语境，例如工程菌、重组表达、代谢工程、生产、发酵、制备、产率、放大或纯化。
- 普通疾病诊断、单纯治疗用途、基础细胞研究、传统酿酒/酱油/泡菜发酵、天然提取和纯化学合成，不因出现“细胞、酶、发酵、基因”等词就自动进入生物制造样本。
- 生物医药仅在涉及重组表达、细胞培养制造、载体生产、CAR-T制备、发酵或下游生物工艺时纳入；单纯药物作用机制或治疗方法不纳入。

### 2. 最终分为1—5分，不设置离散相关性标签

| 分值 | 含义 | 典型技术 |
|---:|---|---|
| 5 | 生物制造核心范式或前沿制造路线 | 生物制造、合成生物学、DBTL、细胞工厂、虚拟细胞、分布式制造、电生物合成 |
| 4 | 关键底层工具、菌株/代谢/酶工程和规模化能力 | CRISPR、代谢工程、酶定向进化、高通量筛选、发酵工程、中试放大 |
| 3 | 重要生产工艺、装备和较高相关应用路线 | 生物反应器、分离纯化、固定化酶、非粮生物质、DNA存储、生物安全 |
| 2 | 生物制造产品和应用类别 | 生物基材料、平台化学品、生物燃料、微生物蛋白、工业酶、重组药物 |
| 1 | 高通用度的终端消费或配方原料应用 | 个护活性原料、香料、色素、功能糖等 |

最终 `biomanufacturing_score` 取有效技术类型的最高核心度，范围为1—5；`biomanufacturing_score_raw` 是所有有效规则的累计证据分，用于排序和复核，不代表技术核心程度。

典型判断如下：

- “基于合成生物学DBTL循环构建微生物细胞工厂”——5分。
- “工程菌通过代谢工程生产丁二酸”——4分。
- “采用CRISPR改造大肠杆菌并发酵生产目标化合物”——4分。
- “CRISPR-Cas9编辑细胞用于治疗肿瘤”——缺少制造语境，不纳入。
- “一种啤酒酿造发酵罐”——传统食品发酵消歧后归零。
- “工程菌发酵生产透明质酸”——按发酵/菌种生产技术计4分；“透明质酸化妆品组合物”不纳入。
- “电生物合成将CO₂转化为乙酰辅酶A”——5分。

默认 `MIN_BIOMANUFACTURING_SCORE = 1`，保留所有通过上下文确认的外围应用专利；如需更保守的样本，可改为2或3。

## 四、公开补充来源及其作用

1. [中国科学院天津工业生物技术研究所：合成生物学与微生物制造工程](https://tib.cas.cn/kxyj/lyfx/202410/t20241031_7412324.html)：补充工业底盘细胞模型、机理模型与AI双驱动菌种设计、DNA大规模合成、细胞工厂优化，以及生物质、合成气和CO₂利用。
2. [中国科学院天津工业生物技术研究所：生物设计中心](https://tib.cas.cn/rcdw/rczp/zyjszp/t_8227670.html)：补充工业酶/菌种数据库、细胞模型、自动化菌种改造、生物制造数据中心和数据—模型—应用飞轮。
3. [中国科学院天津工业生物技术研究所：虚拟细胞相关项目](https://www.tib.cas.cn/xwdt/mtsm/t_8142486.html)：补充虚拟细胞、AI与机理模型融合及模型引导的生物设计。
4. [海通国际《把握合成生物发展趋势，聚焦产业链上下游突破》](https://pdf.dfcfw.com/pdf/H3_AP202411081640790019_1.pdf)：补充原料层、工具层、技术平台层和应用层，以及非粮碳源、一碳原料、DNA读写编辑和发酵技术。
5. [华安证券《生物制造大有可为，助力新一波产业革命浪潮》](https://pdf.dfcfw.com/pdf/H3_AP202503111644268443_1.pdf)：补充高效表达平台、食品级菌株平台、表达元件库、阳性转化子筛选和蛋白高效表达。
6. [公开合成生物学产业深度报告](https://pdf.dfcfw.com/pdf/H3_AP202407301638457775_1.pdf)：补充DBTL测试/学习、高通量与自动化筛选、多组学和规模放大。
7. [公开生物制造工艺与产业链研究](https://pdf.dfcfw.com/pdf/H3_AP202404021629430868_1.pdf)：补充摇瓶—小试—中试—量产放大、pH/溶氧/补料/搅拌控制及分离纯化。

科研机构资料主要用于确认技术范式和前沿方向；资本市场报告主要用于补充产业化、工艺放大、设备和专利常用表述。具体公司、产品品牌和阶段性宣传词原则上不进入词典，少量模型名或技术名仅在已经形成通用技术指代时保留。
